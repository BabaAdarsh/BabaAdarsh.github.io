# babaadarsh.github.io

Academic website of Adarsh Kumar — Jekyll, hosted free on GitHub Pages.

## Publish (one-time, ~5 min)

1. Create a public repo named exactly **BabaAdarsh.github.io** at https://github.com/new
2. Click "uploading an existing file" and drag in all CONTENTS of this folder (including `_layouts`, `_posts`, `assets`, `blog`).
3. Commit. Site is live at https://babaadarsh.github.io in ~2 minutes.

## Update

- New blog post: add `_posts/YYYY-MM-DD-title.md` (copy front-matter from existing post)
- Photo: upload square image as `assets/img/profile.jpg`
- CV PDF: upload as `assets/cv/Adarsh_Kumar_CV.pdf`
- Scholar/ORCID/LinkedIn: fill URLs in `_config.yml`
